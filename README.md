# Meu Projeto Front-End

## Descrição
Este é um projeto iniciante de Front-End desenvolvido para praticar habilidades básicas em HTML, CSS e JavaScript. O objetivo principal é criar uma interface simples e funcional, aprendendo conceitos essenciais de desenvolvimento web.

## Funcionalidades
- Navegação simples entre seções.
- Botões interativos com efeitos visuais.
- Estrutura organizada de pastas para HTML, CSS e JS.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
